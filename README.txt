MOODLE REPORT usersbadges
============================

A Moodle admin report. It show the list of user with badges and the badges list from each user.

Package tested in: moodle 3.0+

QUICK INSTALL
==============
Download zip package, extract the usersbadges folder and upload this folder into report/.
